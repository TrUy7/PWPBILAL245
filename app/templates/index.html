Aplikasi Management User

Ini merupakan implementasi project yang diberikan di kelas

Features

1

Halaman Registrasi

Form registrasi untuk pengguna baru dengan hashing password untuk keamanan data.

2

Halaman Login

Form login pengguna dengan sesi untuk menyimpan status login.

3

Halaman Dashboard

Menampilkan daftar pengguna, dengan opsi untuk menambah, mengedit, atau menghapus data.

4

Fitur CRUD

CRUD penuh untuk menambahkan, mengedit, dan menghapus pengguna dengan validasi.

5

Halaman Logout

Logout pengguna secara aman dan hapus sesi dengan satu klik.

6

Model Database

Tabel pengguna dengan kolom ID, username, role, email, dan password_hash.
